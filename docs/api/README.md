# API Docs

Documentation for integrating with the Graph Change Detector service. These guides cover API usage, performance considerations, testing flows, and troubleshooting tips. They reference the core architecture specs in `../architecture/graph/` for background on change detection and schema design.

Available documents:

- `graph_change_detector_api.md` – endpoints, payload shapes, and auth expectations.
- `graph_change_detector_usage.md` – common request flows and sample calls.
- `graph_change_detector_performance.md` – guidance for scaling and tuning.
- `graph_change_detector_testing.md` – test strategies and fixtures.
- `graph_change_detector_troubleshooting.md` – debugging checklist and known issues.
